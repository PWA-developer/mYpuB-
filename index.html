Información sobre la aplicación

mYpuB es una aplicación web progresiva para subir, compartir y gestionar imágenes y vídeos.
¿Para qué sirve?
Permite que cualquier usuario suba imágenes o vídeos y decida si desea compartirlos públicamente (comentable, descargable por todos los usuarios) o mantenerlos privados.
¿Cómo utilizarla?
1. Regístrate usando un correo de Gmail y una contraseña segura.
2. Sube tus imágenes o vídeos desde el módulo "SUBIR TU".
3. Visualiza y descarga los archivos compartidos en "GALERÍA".
4. Los administradores/desarrolladores pueden gestionar usuarios y contenidos.
